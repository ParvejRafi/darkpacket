# Certificate Images

Place your certificate images in this folder with the following names:

1. **jr-pentester.jpg** - Jr Penetration Tester (TryHackMe)
2. **cybersecurity-intern.jpg** - Cybersecurity Intern Certificate
3. **cybersecurity-awareness.jpg** - Cybersecurity Awareness
4. **google-cybersecurity.jpg** - Google Foundations of Cybersecurity
5. **linux-foundation.jpg** - Linux Foundation LFS101
6. **pcep-python.jpg** - PCEP Python Programmer
7. **api-security.jpg** - API Security Fundamentals

## Image Requirements:
- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 800x600px or similar aspect ratio
- **File Size**: Keep under 500KB for optimal loading

## How to Add Verification Links:

In the `index.html` file, update the `href="#"` in each certificate card's verify button with your actual verification URL:

```html
<a href="YOUR_VERIFICATION_URL_HERE" target="_blank" class="cert-verify-btn">
```

For example:
```html
<a href="https://tryhackme.com/certificate/YOUR_CERT_ID" target="_blank" class="cert-verify-btn">
```
